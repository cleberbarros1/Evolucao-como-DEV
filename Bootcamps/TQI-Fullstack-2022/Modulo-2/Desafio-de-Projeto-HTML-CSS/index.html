TQI Fullstack Developer - 2022_

O Bootcamp TQI Fullstack Developer é um programa gratuito para formação em desenvolvimento fullstack e recrutamento de desenvolvedores juniores pela TQI. O programa apresenta cursos, desafios práticos e mentorias em tecnologias essenciais para front-end e back-end, além de banco de dados e introdução à Quality Assurance (QA) para testes e qualidade. _
